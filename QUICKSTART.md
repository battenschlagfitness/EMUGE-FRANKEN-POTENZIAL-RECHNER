# ⚡ Potenzial-Kalkulator – Quick Start

## 🎯 Sofort loslegen (30 Sekunden)

### Schritt 1: App öffnen
Öffne die Datei **`potenzial-calculator.html`** in deinem Browser.

Das war's! ✅ Die App ist sofort einsatzbereit.

---

## 📱 Auf dem Handy nutzen

### iOS (iPhone/iPad)
1. Öffne **Safari**
2. Öffne die HTML-Datei (z.B. vom Cloud-Speicher oder E-Mail-Anhang)
3. Tippe auf **Teilen** (↗️-Symbol unten rechts)
4. Wähle **Zum Home-Bildschirm**
5. Fertig! Die App ist nun als Symbol verfügbar

### Android
1. Öffne **Chrome** (oder einen anderen Browser)
2. Öffne die HTML-Datei
3. Tippe das **Menü** (⋮) oben rechts
4. Wähle **„Zum Home-Bildschirm hinzufügen"**
5. Fertig! Die App ist nun als Symbol verfügbar

---

## 📝 Erste Berechnung starten

### Schritt 1: Kundenname eingeben
```
Kundenname: "Musterfirma GmbH"
```

### Schritt 2: Maschinen ODER Mitarbeiter eingeben (eines ist erforderlich)
```
Option A: CNC-Maschinen: 15
oder
Option B: Mitarbeiter: 75
```

### Schritt 3: Branche auswählen
```
→ Maschinenbau / Lohnfertigung / Aerospace / Werkzeug-/Formenbau
```

### Schritt 4: Speichern
```
Klick auf "💾 Speichern"
```

**Fertig!** Die Berechnung wird angezeigt und der Kunde ist gespeichert. ✅

---

## 🔄 Gespeicherte Kunden verwalten

### Kundendaten ändern
1. Klick auf die Kundenkarte
2. Ändere die Werte
3. Klick auf "💾 Aktualisieren"

### Kunde löschen
1. Klick auf **🗑️** auf der Kundenkarte
2. Bestätige die Löschung

---

## 🧮 Was wird berechnet?

```
Jahrespotenzial = 
  Maschinen × 5.000 € × Schichtfaktor × Materialfaktor × Branchenfaktor
```

**Beispiel:**
- 12 Maschinen
- 2 Schichten (1,7×)
- Stahl (1,0×)
- Lohnfertigung (1,1×)

= 12 × 5.000 × 1,7 × 1,0 × 1,1 = **112.200 €/Jahr**

## 🆘 Probleme?

### Daten sind verschwunden
→ Browser-Cache wurde geleert? Browser-Speicher (LocalStorage) wurde gelöscht?
→ Leider nicht mehr zu retten, aber du kannst die App neu verwenden

### App lädt nicht
→ Versuche einen anderen Browser (Chrome, Firefox, Safari)
→ Stelle sicher, dass JavaScript aktiviert ist

### Falsche Berechnungen
→ Checke, ob die richtigen Parameter eingegeben sind
→ Siehe „ANLEITUNG.md" für Details zur Formel
